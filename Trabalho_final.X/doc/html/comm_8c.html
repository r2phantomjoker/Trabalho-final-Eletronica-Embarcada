◆ MAX7219_Write()

void MAX7219_Write ( uint8_t address,
uint8_t data )

Envia um pacote de 16 bits para o driver MAX7219 via SPI.

Observação
O MAX7219 exige a seguinte sequência rigorosa:
  1. CS Low, Habilita comunicação.
  2. Envia 8 bits de Endereço, MSB.
  3. Envia 8 bits de Dados, LSB.
  4. CS High (Borda de subida carrega os dados no registrador interno - Latch).
Parâmetros
addressEndereço do registrador do MAX7219.
dataValor a ser escrito no registrador.

◆ UART_EnviaDados()

void UART_EnviaDados ( void )

Transmite o pacote de telemetria do sistema via UART.

Coleta os estados globais do sistema e envia via telemetria.

Observação
Protocolo do Pacote: "$A,D,M,PPP,VV.V,TT.T\r" Onde:
  • $: Cabeçalho de início de frame.
  • A: Andar Atual (0-9)
  • D: Andar Destino (0-9)
  • M: Estado do Motor (0-9)
  • PPP: Posição em mm (000-999)
  • VV.V: Velocidade (00.0-99.9)
  • TT.T: Temperatura (00.0-99.9)
  • \r: Terminador (Carriage Return)

Tabela de inicialização e configuração do driver MAX7219.

Tabela contendo a sequência de comandos de inicialização do MAX7219.

  • Observação
    Estrutura do vetor: Pares de bytes ordenados [Endereço do Registrador, Dado]. A função MatrizInicializa() percorre este vetor de 2 em 2 bytes.

Configurações aplicadas:

  • 0x09 (Decode Mode): 0x00 - Controle individual dos LEDs (Matriz).
  • 0x0A (Intensity): 0x00 - Brilho mínimo.
  • 0x0B (Scan Limit): 0x07 - Habilita todas as 8 linhas/dígitos.
  • 0x0C (Shutdown): 0x01 - Sai do modo de economia de energia.
  • 0x0F (Display Test):0x01 - Pisca a tela para reset visual.
