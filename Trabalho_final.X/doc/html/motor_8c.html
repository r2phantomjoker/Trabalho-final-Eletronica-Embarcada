void SENSORES_CalcularVelocidade ( void )

Calcula Velocidade e Posição lendo o Hardware (TMR0).

Callback de baixa frequência para cálculos físicos. Lê o registrador de hardware TMR0 (Encoder), calcula a velocidade e atualiza a posição.

Observação
Deve ser chamada periodicamente (ex: Timer 4 a cada 100ms).
